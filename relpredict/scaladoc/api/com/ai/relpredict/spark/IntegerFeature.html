case class IntegerFeature(name: String, desc: String, parms: Map[String, String], ss: SparkSession, df: DataFrame, column_map: Datamap) extends Feature[Long] with Product with Serializable

An IntegerFeature is represented by a set of Integers. They can be normalized through their encode parameter. Integers are represented by a Long to cover short, int, and bigint data types.
