Related Doc: package spark

abstract class Feature[A] extends Serializable

A Feature is an abstract class representing a variable to be encoded and used for training and prediction. It is sub-typed through several case classes to provide functionality. A Feature is mapped by default to a column with its own name. A Feature can be mapped to a different name using a column map. A Feature can be transformed using a Datamap.
