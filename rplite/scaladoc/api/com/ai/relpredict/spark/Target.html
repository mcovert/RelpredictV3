Target

Related Doc: package spark

abstract class Target[A] extends Serializable

A Target represents a variable to be predicted. It is typed. A Target is mapped to a column, which by default is the same as its name. A Target (and all of its specified Algorithms) is assigned to a FeatureSet. If multiple algorithms that use different encodings are to be used on a single target, this can be accomplished by duplicating the target (and giving it a different name), assigning each to its required FeatureSet, and then mapping each Target to the same DataFrame column (using a column map). String Targets can also use Data maps to translate their values.
